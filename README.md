# Music rating website
A resource for music fans to keep up with their favourite music.  

The website uses Laravel Jetstream and the Inertia.js stack (Laravel + Vue.js). Have a look at the documentation here:  
- https://jetstream.laravel.com/2.x/introduction.html
- https://laravel.com/docs/8.x/collections
- https://v3.vuejs.org/guide/introduction.html
- https://inertiajs.com/

The website pulls related artist information from Spotify Web APIs. Have a look at the documentation at:  
- https://developer.spotify.com/documentation/web-api/
